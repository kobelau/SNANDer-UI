# SNANDer-UI
This user interface was developed based on two open-source projects: https://github.com/McMCCRU/SNANDer and https://github.com/Droid-MAX/SNANDer. First of all, our thanks go out to both contributors. The UI aims to provide users with a more convenient operation experience, simplifying the entire process. In its design, particular attention has been paid to optimizing user experience, ensuring that even first-time users can quickly get up to speed.

The software offers 5 trial opportunities so that users can fully understand its functionalities and features. Once these trial uses are exhausted, activation is required to continue using the software. This mechanism is primarily in place to prevent unauthorized resale and usage.

For users who need to activate the software, we offer a convenient solution: You can join our QQ group (group number: 252233911) to obtain free activation services. Please remember, this service is entirely free—free, free, free; important things are said three times! Simply provide your device's machine code (each device has a unique code) to the administrator, and you can easily complete the activation process.

We hope that through this approach, every user in need can enjoy the convenience brought by the software without any barriers.
